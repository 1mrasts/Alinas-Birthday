# Alina's Birthday

A showcase website dedicated to Alina's birthday. The project includes several sections: about me, family, and childhood.

## Tech Stack

- **React** — UI framework for building interfaces
- **TypeScript** — typing for reliability and maintainability
- **Vite** — fast build and development
- **React-Router** — routing between pages
- **Sass** — styling with preprocessor support

## Project Structure

Built using **FSD** (Feature-Sliced Design) methodology:

- `src/pages` — application pages
- `src/widgets` — self-contained UI blocks
- `src/app` — entry point and app configuration

## Sections

- **Main** — main page with core information
- **About Me** — personal details
- **Family** — family section
- **Childhood** — childhood memories
